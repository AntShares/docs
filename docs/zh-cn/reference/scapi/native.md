# 原生合约

Neo3 提供了大量的原生合约，原生合约是 Neo3 中NeoContract组件的一部分。

在 Neo3 中，一切皆合约。对于NEO和GAS这类系统对象，它们也运行在智能合约之中。虽然使用普通的智能合约来构建NEO和GAS也是一个可行的方案，但是采用原生合约来构建的NEO和GAS的运行速度更快。
