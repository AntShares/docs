# Metodo Getconnectioncount

Ottiene il numero corrente di connessioni per il nodo.

## Esempio

Testo della richiesta:

```json
{
   "jsonrpc": "2.0",
   "method": "getconnectioncount",
   "params":[],
   "id": 1
}
```

Testo della risposta:

```json
{
   "jsonrpc": "2.0",
   "id": 1,
   "result": 10
   }
```
